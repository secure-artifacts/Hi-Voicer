# Hi-Voicer 0.2.1 打包测试清单

测试对象：

- NSIS 安装包：`A:\Gmini\Hi-Voicer\src-tauri\target\release\bundle\nsis\Hi-Voicer_0.2.1_x64-setup.exe`
- MSI 安装包：`A:\Gmini\Hi-Voicer\src-tauri\target\release\bundle\msi\Hi-Voicer_0.2.1_x64_en-US.msi`
- 直接运行版：`A:\Gmini\Hi-Voicer\src-tauri\target\release\hi-voicer.exe`

建议优先测试 NSIS 安装包。每项记录：通过 / 失败 / 备注。

## 1. 安装和启动

- 安装包能正常打开、安装、启动。
- 首次启动没有白屏、闪退、明显卡死。
- 主窗口、悬浮按钮、录音波形小窗显示逻辑正常。
- 关闭主窗口后应用行为符合预期，不影响再次打开。
- 旧版本升级安装后，设置、热词、历史记录没有异常丢失。

## 2. 离线和隐私

- 断网后应用仍可打开。
- 已安装模型的情况下，断网后可转录本地音频。
- 诊断页、术语库、字幕编辑器、音频处理不依赖云服务。
- 不输入任何在线账号、API Key 或云端配置。
- 音频处理/转码需要 ffmpeg 时，不会自动联网下载；缺失时应提示本地安装或放置 `ffmpeg.exe`。
- CUDA runtime 缺失时，转录不会自动联网下载，应直接提示或回退 CPU。

## 3. 诊断页

- 本机音频环境能显示默认麦克风。
- 本机音频环境能显示默认系统输出设备，并提示系统声音录制仍需在开始录制时验证 WASAPI loopback。
- ffmpeg 状态显示合理：未安装时提示本地缺失，并列出可放置 `ffmpeg.exe` 的目录或 PATH；放入本地 `engines\ffmpeg` 后显示路径。
- 刷新音频环境诊断按钮可重复点击，不报错。
- 保存诊断报告后，报告包含 GPU/加速状态和本机音频环境。
- GPU/CPU 加速状态显示与当前设置一致。

## 4. 模型和转录

- 设置页选择/安装本地模型后，模型状态变为可用。
- 单个短音频可正常转录。
- 长音频转录不会只按 60 秒硬切，分段结果比旧版自然。
- 输出三种格式正常：纯文本、时间线 TXT、SRT。
- 时间线明确是估算时间线，不伪装成逐字真实时间戳。
- 转录后可进入字幕编辑器。

## 5. 术语库

- 默认分类存在：人名、机构、项目、技术词、常用替换。
- 可新增、重命名、删除分类。
- 删除分类后，该分类下规则会移动到有效分类。
- 可新增、编辑、启用/停用、删除术语规则。
- 旧热词 JSON 数组可导入，不会被误当成分类。
- 新版术语库 JSON 可导入/导出，分类和规则都保留。
- 规则指向不存在分类时，会回落到有效分类。

## 6. 录制

- 只录麦克风：能生成有效 WAV，电平有反应。
- 只录系统声音：播放系统声音时能录到内容。
- 麦克风 + 系统声音：能保存双轨；转写模式能使用混音文件。
- 录制中波形窗口显示/隐藏正常。
- 停止录制后不会残留损坏的空文件；太短或空录音应提示重录，不显示保存成功。
- 使用全局快捷键停止录制时，如果录音太短、空录音或保存失败，主窗口应显示错误提示，不应只在后台日志里静默失败。
- 无模型时，音频保存模式仍可用；转写模式会提示先配置离线模型。

## 7. 字幕编辑器

- 音频能播放。
- 点击字幕能跳转到对应音频位置。
- 搜索字幕文字后，点击结果能定位音频。
- 可修改字幕文本。
- 可调整起止时间，时间不会变成 NaN 或重叠异常。
- 可拆分字幕；太短字幕不会拆出空段。
- 可合并下一条字幕，英文/数字相邻时空格合理。
- 可导出纯文本、时间线 TXT、SRT。
- 可导出选中字幕对应 WAV 音频段，前后余量不切字头字尾。
- 可勾选多条字幕并一次导出多段 WAV 音频。
- 字幕片段默认导出到原始音频所在目录；选择自定义片段目录后，应导出到自定义目录。

## 8. 基础音频处理

- 未选文件时，处理按钮不可用。
- 选择文件失败时有明确提示。
- 可拖放单个文件和多个文件加入处理队列。
- 可选择整个文件夹，递归加入支持的音频/视频文件。
- 可批量处理队列里的多个文件，并在下方显示每条处理状态。
- 可清空处理队列和历史结果。
- 默认输出到每个原始文件所在目录；选择自定义输出目录后，应输出到自定义目录。
- 音量标准化能生成可播放 WAV。
- 静音裁剪能生成可播放 WAV。
- 人声基础增强能生成可播放 WAV。
- 电流声削弱预设能生成可播放 WAV。
- 每条处理完成结果下方都有独立播放器；点击该条“试听”能播放对应效果音频。
- 切换到其他功能再返回音频处理页，已处理结果仍保留，直到手动清空历史。
- 处理失败或超时后，不应留下明显半成品输出文件。
