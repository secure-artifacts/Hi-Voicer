# Hi-Voicer 0.2.1 打包测试清单

测试对象：NSIS 安装包、MSI 安装包和直接运行版。

## 1. 安装和启动

- 安装包能正常打开、安装、启动。
- 首次启动没有白屏、闪退、明显卡死。
- 主窗口、悬浮按钮、录音波形小窗口显示逻辑正常。
- 旧版本升级安装后，设置、热词、历史记录没有异常丢失。

## 2. 离线和隐私

- 断网后应用仍可打开。
- 已安装模型的情况下，断网后可转录本地音频。
- 诊断页、术语库、字幕编辑器、音频处理不依赖云服务。
- 不输入任何在线账号、API Key 或云端配置。
- 音频处理/转码需要 ffmpeg 时，不会自动联网下载；缺失时应提示本地安装或放置 `ffmpeg.exe`。
- CUDA remains removed and no GPU runtime is downloaded automatically. CPU is the default stable path; DirectML is experimental and must pass the Diagnostics PoC plus CPU vs DirectML benchmark.

## 3. 诊断页

- 本机音频环境能显示默认麦克风。
- 本机音频环境能显示默认系统输出设备，并提示系统声音录制仍需在开始录制时验证 WASAPI loopback。
- ffmpeg 状态显示合理：未安装时提示本地缺失，并列出可放置 `ffmpeg.exe` 的目录或 PATH。
- 刷新音频环境诊断按钮可重复点击，不报错。
- 保存诊断报告后，报告包含 CPU 识别运行时状态和本机音频环境。
- CPU 识别运行时状态显示与当前设置一致。

## 4. 模型和转写

- 设置页选择/安装本地模型后，模型状态变为可用。
- 单个短音频可正常转写。
- 长音频转写不只按 60 秒硬切，分段结果自然。
- 输出三种格式正常：纯文本、时间线 TXT、SRT。
- 转写后可进入字幕编辑器。

## 5. 术语库

- 默认分类存在：人名、机构、项目、技术词、常用替换。
- 可新增、重命名、删除分类。
- 可新增、编辑、启用、停用、删除术语规则。
- 新版本术语库 JSON 可导入/导出。

## 6. 录制

- 只录麦克风：能生成有效 WAV，电平有反应。
- 只录系统声音：播放系统声音时能录到内容。
- 麦克风 + 系统声音：能保存双轨；转写模式能使用混音文件。
- 录制中波形窗口显示/隐藏正常。
- 停止录制后不留下损坏空文件；太短或空录音应提示重录。

## 7. 字幕编辑器

- 音频能播放。
- 点击字幕能跳转到对应音频位置。
- 可修改、拆分、合并字幕。
- 可导出纯文本、时间线 TXT、SRT。
- 可导出选中字幕对应 WAV 音频段。

## 8. 基础音频处理

- 未选文件时，处理按钮不可用。
- 可拖放单个文件和多个文件加入处理队列。
- 可批量处理队列里的多个文件，并显示每条处理状态。
- 音量标准化、静音裁剪、人声增强、格式转换、音频合并能生成可播放文件。
- 处理失败或超时后，不应留下明显半成品输出文件。
